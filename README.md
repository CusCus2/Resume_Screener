# Resume_Screener
This is a project that takes a candidates CV in PDF format and the job description, and passes both through an LLM to determine if the candidate should move to the next round. This project was built using python and ollama

## How To run
simply configure the file path in execution.py to match the location of your selected CV,
And editi the job_description text in execution.py.

then simply run the execution.py script;
py execution.py

to install the necessary requirements run the following comand in the terminal:
pip install -r requirements.txt
